# Ecotrip

Projeto desenvolvido na formação GitHub Copilot da [dio.me].

A ferramente realiza o cálculo das emissões de CO² baseadas no método de transporte informado.

Algumas rotas estáticas foram pré-definidas no projeto e existe também a possibilidade de informar manualmente a distância para a rota.

Para critérios de desenvolvimento foram inseridos locais específicos, porém a aplicação é flexivel para inserir mais localidades.
